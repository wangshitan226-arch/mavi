# MAVI
> 一个“教学过程可视化＋学习过程资产化”的系统
---
## 项目背景
在两段家教背景和一段机构背景的数学教育经历中，我发现现有的大学生家教存在以下问题
- 家教/机构教学效果**不透明**，家长和学生往往花钱了但不知道到底进步了什么，老师也难以用数据有交代。
- 家教/机构的老师入门门槛低，教学多**不成体系**，质量参差不齐，很容易产生**知识漂移**

本项目希望通过一个从老师端作为切入点的系统的实现，来解决上述问题
---

## 项目目标
- 实现教学过程可视化，向家长证明孩子真的进步了
- 以最低的门槛，实现让老师上手就有成体系的教学路线，并配以可视化教学工具，助力教学
- 不断地回归教材，不断地丰盈档案，保证知识不漂移，实现学习过程资产化

## 项目功能
- [] 教材基座 + 动态知识结构树
- [] 错因资产引擎 + 演化路径
- [] 可视化讲解资产库（Manim集成）
- [] 班级/学生匿名档案 + 效果趋势可视化
- [] 整卷/阶段性诊断报告导出
- [] 个性化教学调整建议
- [] 长期成长档案导出
